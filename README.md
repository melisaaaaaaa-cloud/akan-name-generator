
# Akan Name Generator

## Description
This is a web application automatically gives a user their name based on the day they were born and gender.

## Author

Melissa Muthoni

## Setup Instructions

To run this project locally:

1. Clone or download this repository
2. Make sure both index.html and style.css are in the same folder
3. Place any photos inside an images/ folder in the same directory
4. Open index.html in any web browser — no installs needed!
